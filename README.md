# ScrollSaver

Scroll saver is a chrome extension that allows users to create bookmarks for scroll positions on a website

After clicking on ScrollSaver in the extensions tab on Chrome, you can click "Create Bookmark" to set a scroll position on the page. After creation, you can click "Go" on it to jump to that position on the page or click "Delete" if you would like to delete a position. You can also double click on a position to rename it. 

![Example](/ScrollSaver/images/examplewebsite.png)

## All Bookmarks

The ScrollSaver popup only contains the saved scroll positions for the current page. If you want to see all your bookmarks, click "View All Bookmarks"

That button will take you to a page to see all your bookmarks. You can search by keyword for a your bookmarks if you have many. 

![AllBookmarks](/ScrollSaver/images/allbookmarks.png)

## PDF Functionality

ScrollSaver also works for locally stored PDFs opened in Chrome. If you have a PDF open in Chrome, a new green button titled "Open in ScrollSaver" will appear in the ScrollSaver extension popup. Once you click that, you will be able to place bookmarks on your pdf and jump to them as you please
